## Run Locally.
### First clone the repo
```bash
git clone https://github.com/praneethgarre/ScolarshipFinder.git
cd ScholarshipFinder
```

### Run backend.
```bash
cd backend
npm i 
npm start
```
### Run frontend.
Then open a new terminal and
```bash
cd scholarship-finder
npm i
npm run dev
````
